Centos连接无线网

By yanlc

发表于 2015-07-26

文章目录

今天将一台老电脑装了Centos系统，安装完之后是命令行界面，由于安装的时候没有配置网络环境，ifconfig命令没有需要从新装，首先配置有限网络

cd /etc/sysconfig/network-scripts/ 用vi打开里面的第一个文件，编辑内容
HWADDR=00:0C:29:8D:24:73
TYPE=Ethernet
BOOTPROTO=static #启用静态IP地址
DEFROUTE=yes
PEERDNS=yes
PEERROUTES=yes
IPV4_FAILURE_FATAL=no
IPV6INIT=yes
IPV6_AUTOCONF=yes
IPV6_DEFROUTE=yes
IPV6_PEERDNS=yes
IPV6_PEERROUTES=yes
IPV6_FAILURE_FATAL=no
NAME=eno16777736
UUID=ae0965e7-22b9-45aa-8ec9-3f0a20a85d11
ONBOOT=yes #开启自动启用网络连接
IPADDR0=192.168.0.168 #设置IP地址
PREFIXO0=24 #设置子网掩码 不要设置 255.255.255.0
GATEWAY0=192.168.0.1 #设置网关
DNS1=8.8.8.8 #设置主DNS，不要设置自己所在区的dns，我配置过，不行
DNS2=8.8.4.4 #设置备DNS

配置完之后重启网络服务

service network restart

如果网络没有连上 reboot重启

安装ifconfig

yum install -y net-tools*

安装完之后 ifconfig查看网卡信息，发现只有一个有线网卡enp8s0和一个回路lo
查啊查，发现要NetworkManager服务没有开，泪奔

service NetworkManager start

启动服务，终于多出了一块无线网卡，以防万一，确保此网卡接口启动执行命令

sudo ip link set wlan0 up

扫描附近的无线网络

sudo iw dev wlan0 scan | less

连接没有加密的网络

sudo iw dev wlan0 connect [网络 SSID]

连接没有加密的网络wep加密网络

sudo iw dev wlan0 connect [网络 SSID] key 0:[WEP 密钥]

连接WPA或WPA2协议的话比较麻烦，需要修改/etc/wpasupplicant/wpa_supplicant.conf文件，增加如下行

network={
ssid=”网络 ssid”
psk=”密码”
priority=1
}

配置文件修改完成后

sudo wpa_supplicant -i wlan0 -c /etc/wpa_supplicant/wpa_supplicant.conf

这里发现了冲突，google了一下发现wpa_supplicant和NetworkManager不能同时启动，否则会冲突，只能关闭NetworkManager

service NetworkManager stop

然后在

sudo ip link set wlan0 up
开启无线网卡

获取 IP 地址

sudo dhcpcd wlan0
